# Quick Start

### Add default.json file under conifg folder as below:

```
{
  "mongoURI": "your-mongoDB-atlas-credentials",
  "jwtSecret": "your-secret-token"
}
```

### Install server dependencies

```
npm install
```

### Install client dependencies

```
cd client
npm install
```

### Run from root level

```
npm run dev
```

### Check browser on

http://localhost:5000/
